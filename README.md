# sash shell

## How to run?
Compile - `make main`
Run     -  `./a.out`  

## Built in commands:
 -> cd
 
 -> ls
 
 -> pwd
 
 -> echo
 
 -> jobs
 
 -> pinfo
 
 -> overkill
 
 -> kjob


 Executes foreground and background processes and informs about the exit status of the command. Now added support for pipes and added many builtin commands to handle jobs
